■星界機神ストラトノヴァ
ジャンル：スキルレールシューティング
プレイ人数：1人
制作人数：1人
基本システム：
プレイヤーは戦闘機を操作して星に向かって移動する
敵が星から向かってくるので攻撃したりスキルを使って撃退していく。

・プレイ中に敵を見やすくするためにライトの位置を変更しました。

・ポストプロセスを使用して、戦闘機の噴射エフェクトやレーザーの明るさなどを変えて見た目をきれいにしました。

・タイトルシーンにいろいろ動きを付けて少しでも楽しめるようにしました。
